# -6-SQL-JOINs.-1-Patient-Level-Dashboard.-Zero-Guesswork.
Most dashboards show you numbers. This one shows you the patient.  I built a Hospital Patient Analytics Dashboard where every single record — diagnosis, doctor, admission date, charges, medicine history — is connected through 6+ SQL tables (patients, appointments, surgery, bills, beds, rooms, department).



The hardest part wasn't Power BI. It was the SQL.

I wrote complex LEFT JOIN queries to merge fragmented hospital data into one clean, queryable structure — then transformed and cleaned it before a single visual was built.

🔍 What it does:
→ Click any patient → instantly see their full medical journey
→ Doctor assigned, diagnosis, AMT paid, satisfaction rating
→ Medicine sales trends & charge-type breakdown
→ Admission-to-discharge timeline tracking

This is what happens when SQL and Power BI work together — not just charts, but a real decision-support tool.

🛠️ Built with: MySQL | Power BI | DAX | Data Modeling | Star Schema
